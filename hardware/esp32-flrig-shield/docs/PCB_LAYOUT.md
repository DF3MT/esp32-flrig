# PCB layout — compact 2-layer (78 × 50 mm)

Rev **D** placement: minimum area, connectors on board edges for cable access.

## Board

| Parameter | Value |
|-----------|--------|
| Layers | **2** (F.Cu + B.Cu), 1.6 mm FR4 |
| Outline | **78 × 50 mm** (`Edge.Cuts` 2…80 mm × 2…52 mm) |
| GND pour | `GND_ESP` on F.Cu inside outline |

## Connector edges (plug access)

```
        FRONT (Y ≈ 6 mm) — radios plug in here
  ┌──────────────────────────────────────────┐
  │  J5 USB-B    J6 USB-C         J1 RJ45    │
  │                                          │
L │  J2 CYD      U5 Hub    U3/U4 I2S         │  logic / power
E │  2×10        U2 LDO    U1 ISO            │
F │                                          │
T │         J3 ESP32-S3 host    J7 Rotor     │  BACK (headers / harness)
  └──────────────────────────────────────────┘
```

| Ref | Position (mm) | Role |
|-----|---------------|------|
| **J5** | 18, 6 | USB-A host → radio **B** (isolated USB planned) |
| **J6** | 44, 6 | USB-A host → radio **C** |
| **J1** | 62, 6 | RJ45 → radio **A** (CAT + line audio) |
| **J2** | 6, 28 | 2×10 → **CYD** (CAT, I2S, 3V3, 5V) |
| **J3** | 28, 34 | 1×8 → **ESP32-S3** OTG + rotor signals |
| **J7** | 54, 34 (90°) | 1×6 → rotor cable (parallel J3 pins 5–8) |

## Logic block (short routes)

| Ref | Position | Notes |
|-----|----------|--------|
| **U2** | 14, 22 | AMS1117 — near J2/J3 power |
| **U5** | 34, 16 | USB2422 — hub between J3 OTG and J5/J6 |
| **U1** | 52, 16 | ISO7741 — between hub and RJ45 CAT |
| **U3** | 22, 36 | PCM5102 — I2S DAC, near J2 |
| **U4** | 36, 36 | INMP441 — I2S mic |

## Silkscreen

Connector roles (**RADIO A/B/C**), **J2/J3/J7** pin hints, and IC references are generated by `kicad/pcb_silk.py` (applied via `apply_compact_layout.py`).

## Regenerate

```bash
cd hardware/esp32-flrig-shield/kicad
python3 apply_compact_layout.py   # placement + silk from layout_compact.py / pcb_silk.py
python3 generate_gerbers.py --validate --zip --preview
```

Source of truth for coordinates: `kicad/layout_compact.py`.
