ESP32 CAT Remote Panel

Firmware direkt aus dem Browser flashen (Chrome, Edge oder Firefox – kein PlatformIO nötig).

Dein Browser unterstützt kein Web Serial. Bitte Chrome, Edge oder Firefox auf Desktop nutzen. Web Flasher braucht HTTPS oder http://localhost – nicht file://.

Nach dem Klick den ESP32 per USB verbinden und den seriellen Port auswählen.

Anleitung

  1. USB-Kabel an den ESP32 (CP2102/CH340).
  2. Board-Variante oben wählen.
  3. Firmware installieren klicken und COM-Port bestätigen.
  4. Schlägt der Upload fehl: BOOT (GPIO0) gedrückt halten, kurz RESET, BOOT loslassen, erneut versuchen.
  5. Nach dem Flash: WiFi-AP ESP32-CAT-Panel / Passwort hamradio123 → http://192.168.4.1

Firmware-Dateien fehlen?

Lokal bauen und Flasher-Assets erzeugen:

./scripts/build_flasher_assets.sh
./scripts/serve_flasher.sh

Dann http://127.0.0.1:8765/ öffnen.

Basiert auf ESP Web Tools. Alternative: ./scripts/flash.sh esp32-cyd
